Installatie
-----------
- installeer Node.js https://nodejs.org/download/
- installeer GitHub Desktop: https://desktop.github.com/
- clone deze repository
- run 'npm install' in de root van de project directory
- run 'npm install -g grunt-cli'

Lokaal builden
--------------

   grunt

Lokaal testen
-------------

   grunt nodeunit
